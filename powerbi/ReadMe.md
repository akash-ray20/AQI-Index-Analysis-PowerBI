# Power BI File

This folder contains the Power BI report file used to build the AQI dashboard.

File:
AQI Index Analysis.pbix

The report includes:
- Data loaded from a cleaned AQI dataset
- A separate Calendar table for date-based analysis
- Relationships between the AQI dataset and the Calendar table
- Measures and calculated columns defined using DAX
- Interactive visuals including KPIs, charts, and maps

The PBIX file represents the final version of the dashboard used for analysis and presentation.
