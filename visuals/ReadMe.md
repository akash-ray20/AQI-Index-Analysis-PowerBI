# Visuals

This folder contains screenshots of the Power BI dashboard used for preview and documentation.

Files:
- dashboard.png
- AQI_guage_state_wise.png
- map_view_and_trend_over_time.png 

## dashboard.png:
Full view of the AQI dashboard showing KPIs, slicers, charts, and map.

## AQI_guage_state_wise.png  
This image shows the KPI section of the dashboard for a selected state (Delhi). It includes:
- Gauge displaying Average AQI
- Maximum AQI value card
- Primary Pollutant card
- AQI Category card
All values update dynamically based on slicer selections.

## map_view_and_trend_over_time.png  
This image shows the analytical visuals of the dashboard, including:
- AQI by State map with dynamic bubble size and color representing AQI severity
- AQI Level legend aligned with standard AQI ranges
- AQI Trend over Time area chart with Year and Month slicers to analyze temporal variations

These visuals together represent the core analytical and storytelling components of the AQI dashboard.
